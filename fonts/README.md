# Polices Plum Living

Ce dossier contient les polices de caractères utilisées sur le site Plum Living.

## Structure

```
fonts/
├── cera/              # Police principale Cera
├── blacklist/         # Police Blacklist (titres)
├── source-sans-3/     # Police Source Sans 3
├── sailec/            # Police Sailec
├── canela/            # Police Canela
├── lato/              # Police Lato
└── icons/             # Polices d'icônes
```

## Polices identifiées sur le site

### Cera (Police principale)
- **Usage** : Body text et headings
- **Poids** : 400, 500, 600, 700
- **Style** : Normal

### Cera Pro
- **Usage** : Variante de Cera
- **Poids** : 300, 500, 700
- **Style** : Normal

### Blacklist (Titres)
- **Usage** : Titres et éléments visuels
- **Variantes** : Bold, ExtraBold, SemiBold, Italic
- **Poids** : 400, 500, 800

### Source Sans 3
- **Usage** : Police secondaire
- **Poids** : 400, 500, 600, 700
- **Style** : Normal

### Sailec
- **Usage** : Police d'accompagnement
- **Poids** : 400, 500
- **Style** : Normal, Italic

### Canela
- **Usage** : Police décorative
- **Poids** : 300
- **Style** : Italic

### Lato
- **Usage** : Police utilitaire
- **Poids** : 400, 700
- **Style** : Normal, Italic

### Polices d'icônes
- Icons
- outline-icons
- brand-icons
- swiper-icons

## Instructions

1. Téléchargez les fichiers de polices depuis le Google Drive
2. Placez chaque famille de polices dans son dossier respectif
3. Organisez les fichiers par poids et style (ex: `cera-regular.woff2`, `cera-bold.woff2`)

## Formats supportés

Pour une compatibilité optimale, utilisez les formats suivants :
- `.woff2` (prioritaire)
- `.woff` (fallback)
- `.ttf` (si nécessaire)
