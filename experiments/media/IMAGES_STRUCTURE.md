## Instructions

1. Télécharger les images depuis le Dropbox
2. Les renommer selon la convention choisie (ex: `media-1.jpg`, `media-2.jpg`, etc.)
3. Les placer dans `/images/media/` ou `/images/`
4. Mettre à jour les chemins dans `index.html`

## Convention de nommage recommandée

- **Option 1**: `media-hero.jpg`, `media-1.jpg`, `media-2.jpg`, ..., `media-9.jpg`
- **Option 2**: `0-hero.jpg`, `1.jpg`, `2.jpg`, ..., `9.jpg`
- **Option 3**: Noms descriptifs comme `hero.jpg`, `salle-bain.jpg`, `cuisine-grenat.jpg`, etc.
